Q1: Body and Pre Tags Explanation

Explanation of <body> and <pre> Tags

The <body> tag defines the main content of the HTML document. It contains all the content such as text, images, hyperlinks, tables, lists, etc. It is a required tag in HTML.

The <pre> tag defines preformatted text. Text inside <pre> tag is displayed in a fixed-width font (usually Courier), and it preserves both spaces and line breaks.
